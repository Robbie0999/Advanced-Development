jQuery

Met jQeury kan je lange Js code aan de hand van een methode oproepen. Dit zorgt ervoor dat de webpage sneller laadt. Daarnaast heeft jQuery vele plugins en verhelpt het compatibiliteits problemen met browsers.

JS Infovis

Met deze library kan je gemakkelijk uitgebreide data-analyses maken. Het bevat onder andere HyperTree, Treemap en RGraph. Je kan vele effecten toevoegen aan de hand van plugins zonder al te veel code.

Pixi JS

Deze library is een open source, cross platform, 2D engine waarmee je games en animations kan maken. Pixi is een supersnelle rendering-engine en gebruikt WebGL voor zijn prestaties. Disney gebruikt onder andere deze library.

Three.js

Deze library is een cross-browser JS-bibliotheek waarmee 3D animaties kunnen worden gemaakt. Three.js vertrouwt op WebGL in plaats van normale browser-plug-ins. Three.js is maar een file met een source code op GitHub.

React.js

React gebruikt de JS-bibliotheek om gebruikersinterfaces voor webapplicaties te bouwen Met behulp van de bestaande stukjes code. React wordt beheerd door Facebook.

Vue.js

Vue js is gebaseerd op het Virtual DOM-model en heeft een op componenten gebaseerde architectuur. Met behulp van sjablonen van Vue.js kunnen apps sneller worden gemaakt. Daarnaast heb je minder code nodig.

Modernizr

Modernizr zorgt voor functiedetectie. Het vertelt of de browser JS-, HTML- en CSS-functies heeft. Dit helpt bij het kennen en vermijden van de niet-ge√Ømplementeerde functies.

Verge3D

Door de Verge3D-toolkit toe te voegen, kunnen ontwikkelaars prachtige 3D-visualisaties op websites maken, bijvoorbeeld puzzels, gebruikersinterface en website-indelingen, animaties, AR / VR en nog veel meer

Anime.js

Anime.js is een animatiebibliotheek die animaties zo eenvoudig maakt. De library wordt geleverd met een krachtige API. Je kunt verschillende eigenschappen van CSS op hetzelfde element instellen.

QUnit

QUnit is een tool voor het testen van JavaScript-code. De meeste jQuery-projecten gebruiken QUnit. QUnit is essentieel geworden omdat JS nu een integraal onderdeel is van elk webproject.
